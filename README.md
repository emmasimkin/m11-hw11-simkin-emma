# Assignment 11: Use a CSS and JavaScript Framework to Add Interactivity to a Website

## Background
In this assignment, I used [Bootstrap 5](https://getbootstrap.com/docs/5.3/getting-started/introduction/) to create a single webpage about myself, my hobbies, interests, and career goals. This allowed me to practice using a CSS framework to build responsive, styled layouts efficiently while minimizing custom CSS.

## Components Used
- **Navbar**: A mobile-responsive navigation bar that collapses on smaller screens.
- **Grid System**: Responsive layout for columns in About Me, Hobbies, and Career Goals sections.
- **Spacing Utilities**: Margin and padding classes (`mb-4`, `mb-3`, `p-3`) for layout spacing.
- **Cards**: Used text-only cards for the About Me, Hobbies, and Career Goals sections to organize content.
- **Buttons**: Added a “Learn More” button in the About Me section for interactivity.

## Notes
While completing this assignment, I focused on maintaining mobile responsiveness and professional styling. The Bootstrap components made it straightforward to implement a functional navbar, a structured grid layout, and organized content with text-only cards. The use of cards and buttons adds interactivity and visual organization without relying on images. This exercise reinforced how powerful CSS frameworks can be for creating polished, responsive websites efficiently.
