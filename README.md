# Assignment 11: Use a CSS and JavaScript Framework to Add Interactivity to a Website

## Background
In this assignment, I used [Bootstrap 5](https://getbootstrap.com/docs/5.3/getting-started/introduction/) to create a single webpage about myself, my hobbies, interests, and career goals. This allowed me to practice using a CSS framework to build responsive, styled layouts efficiently with minimal custom CSS.

## Components Used
- **Navbar**: A mobile-responsive navigation bar that collapses on smaller screens.
- **Grid System**: Responsive layout for columns in About Me, Hobbies, and Career Goals sections.
- **Spacing Utilities**: Margin and padding classes (`mb-4`, `mb-3`, `p-4`) for layout spacing.
- **Cards**: Used for Hobbies section to showcase each hobby with an image and description.
- **Buttons**: Added “Learn More” buttons in the About Me section for interactive elements.
